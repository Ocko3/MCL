# MCL - Mount Collection Log

A comprehensive mount tracking addon for World of Warcraft that logs collected mounts in a similar way to Simple Armory.

## About This Fork

This is a **MoP Classic adaptation** of the original MCL addon by Camyana. This version is specifically designed for **WoW Mists of Pandaria Classic (5.4.8)** and includes:

- ✅ MoP-era mounts only
- ✅ Compatibility with MoP Classic API
- ✅ Clean UI for mount tracking
- ✅ Organized by expansion, category, and source
- ✅ Mount collection progress tracking

## Features

- **Mount Organization**: Browse mounts by expansion era and category
  - Mists of Pandaria
  - Cataclysm
  - Wrath of the Lich King
  - Burning Crusade
  - Classic
  
- **Categorized Mounts**: Includes sources such as:
  - Achievements
  - Dungeons & Raids
  - Reputation rewards
  - Quests
  - Rare spawns
  - PvP rewards
  - Professions
  - Racial mounts
  - World events
  - And more!

- **Collection Tracking**: See at a glance which mounts you have collected

- **Mount Details**: View detailed information about each mount including source and acquisition method

## Installation

1. Download or clone this repository
2. Place the `MCL` folder into your WoW MoP Classic AddOns directory:
   - `World of Warcraft/_classic_/Interface/AddOns/MCL/`
3. Reload your UI or restart WoW
4. Type `/mcl` in-game to open the addon

## Usage

- **Toggle Window**: Type `/mcl` or click the addon in your addon compartment
- **Browse Mounts**: Click tabs to view different expansion eras
- **View Details**: Click on a mount to see more information
